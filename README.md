# delphi-web-rad-studio

## Description
This is web server that is used
to connect delphi radstudio.

## Tech stack
- delphi

## Docker stack
- radstudio/paserver

## To run
`sudo ./install.sh -u`
- [Available here](http://localhost:8082)

## To see help
`sudo ./install.sh -h`
